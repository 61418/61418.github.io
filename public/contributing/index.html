61418 exists in order to bring together software engineers who build and maintain open-source software which enhance the AWS Python SDK and general AWS ecosystem.

Accordingly, 61418 wants and needs contributors who are knowledgeable and experienced with AWS services and tools (e.g. boto, AWS CLI, etc.). Due to 61418’s bias for Python, contributors should also exhibit advanced knowledge and skills in Python development, as well as familiar and comfortable with modern tools like uv.
